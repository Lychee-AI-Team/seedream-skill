# Draft: 火山引擎通用API操作Skill

## 需求概述（已确认）
用户想要创建一个skill来操作火山引擎API，要求：
1. ✅ 原则上涵盖全部内容
2. ✅ 过滤旧版接口（只保留最新版本）
3. ✅ 过滤即将废弃/已废弃的接口
4. ✅ 包含全部功能：视觉理解+图像+视频+音频
5. ✅ 具有引导性功能（交互式引导用户）
6. ✅ 这是全新任务，与videoclaw无关

## 接口版本策略
- **Seedance（视频生成）**：只对接 1.5 版本，不对接 1.0
- **Seedream（图像生成）**：只对接 4.0 版本，不对接 3.0
- **其他接口**：需要确认最新版本

## 核心功能范围
1. **图像生成** - 最新版本的文生图、图生图、图像编辑
2. **视频生成** - 最新版本的文生视频、图生视频
3. **任务管理** - 异步任务的创建、查询、列表、删除
4. **其他API** - 需要确认是否需要（视觉理解、音频等）

### 问题2：交互方式设计 ✅ 已确认

**最终选择：C. 混合式**

**关键约束**：AI Agent的模型可能可以识别图片，也可能不能，必须兼容两种可能。

#### 图片输入兼容方案

**场景1：AI Agent支持图片识别（多模态模型）**
```
用户：[上传图片] 帮我把这张图片生成视频
Skill：我看到这是一张宇航员的图片，我来生成视频...
      请描述您想要的镜头运动
      
用户：镜头缓缓拉出，展现全景
Skill：[执行]...完成！
```

**场景2：AI Agent不支持图片识别（纯文本模型）**
```
用户：帮我生成视频，图片URL是 https://example.com/image.jpg
Skill：好的，使用该图片生成视频...
      请描述您想要的镜头运动
      
用户：镜头缓缓拉出
Skill：[执行]...完成！
```

#### 设计原则
1. **图片输入方式支持多种**：
   - 图片URL（网络地址）
   - 本地文件路径（/path/to/image.jpg）
   - Base64编码（直接嵌入）
   - 任务ID引用（使用之前生成的图片）
   
2. **自动检测Agent能力**：
   - Skill启动时检测Agent是否支持图片
   - 根据能力调整引导文案
   
3. **引导文案适配**：
   - 支持图片："您可以上传图片，或提供图片URL"
   - 不支持图片："请提供图片URL或本地路径"

#### 参数传递设计

**简单模式（自然语言）**：
```
生成图片 宇航员在火星
生成视频 用上次生成的图片 镜头拉出
```

**精确模式（命令式）**：
```
生成图片 --prompt "宇航员在火星" --size 1024x1024 --style 电影感
生成视频 --image <task_id> --prompt "镜头拉出" --duration 5 --ratio 16:9
```

**混合模式**：
```
生成图片 宇航员在火星 --style 电影感
生成视频 用图片 https://xxx.jpg 镜头拉出 --duration 5
```
### 问题3：配置管理 ✅ 已确认

**最终选择：混合方案 + 项目级配置**

#### 配置优先级
```
环境变量 > 项目配置 > 全局配置 > 代码默认值
```

#### 配置项清单

**1. 必需配置（环境变量）**：
- `ARK_API_KEY` - 火山引擎API密钥

**2. 可选配置（配置文件/环境变量）**：
- 图像生成默认参数：
  - `VOLCENGINE_IMAGE_RESOLUTION` - 默认分辨率（如 1024x1024）
  - `VOLCENGINE_IMAGE_STYLE` - 默认风格
  - `VOLCENGINE_IMAGE_MODEL` - 默认模型版本
- 视频生成默认参数：
  - `VOLCENGINE_VIDEO_DURATION` - 默认时长（如 5）
  - `VOLCENGINE_VIDEO_RATIO` - 默认比例（如 16:9）
  - `VOLCENGINE_VIDEO_WATERMARK` - 是否添加水印
  - `VOLCENGINE_VIDEO_MODEL` - 默认模型版本
- 音频生成默认参数：
  - `VOLCENGINE_AUDIO_VOICE_TYPE` - 默认语音类型
  - `VOLCENGINE_AUDIO_SPEED` - 默认语速
- 任务管理：
  - `VOLCENGINE_TASK_POLL_INTERVAL` - 轮询间隔（如 10秒）
  - `VOLCENGINE_TASK_TIMEOUT` - 超时时间（如 300秒）

#### 配置文件示例

**全局配置** (`~/.volcengine/config.yaml`)：
```yaml
api:
  # API Key 优先从环境变量读取
  # key: your-api-key

defaults:
  image:
    resolution: 1024x1024
    style: cinematic
    model: doubao-seedream-4-0-t2i-250415
  video:
    duration: 5
    ratio: "16:9"
    watermark: false
    model: doubao-seedance-1-5-pro
  audio:
    voice_type: female
    speed: 1.0
task:
  poll_interval: 10
  timeout: 300
```

**项目配置** (`./.volcengine/config.yaml`)：
```yaml
# 项目特定配置，覆盖全局配置
defaults:
  video:
    ratio: "9:16"  # 竖屏视频
    duration: 10   # 更长时长
```

## 问题4：异步任务处理策略 ✅ 已确认

### 轮询策略

**默认模式：异步返回（选项2）**
```
用户：生成视频 镜头拉出
Skill：任务已创建！任务ID: task-12345
      状态：排队中
      您可以：
      1. 等待完成（我会自动轮询）
      2. 稍后查询：查询任务 task-12345
      3. 继续其他操作
```

**支持模式1：同步等待（用户明确要求）**
```
用户：生成视频 镜头拉出 并等待完成
Skill：[开始生成...轮询中...完成！]
      视频已生成：https://xxx.mp4
```

**支持模式3：后台轮询（如果Agent支持）**
```
用户：生成视频 镜头拉出
Skill：任务已创建！task-12345
      我会在后台监控，完成后通知您
      
[用户继续其他操作...]

Skill：您的视频已生成完成！task-12345
```

### 轮询参数

**默认配置**：
- 轮询间隔：10秒（可通过 VOLCENGINE_TASK_POLL_INTERVAL 配置）
- 超时时间：5分钟（可通过 VOLCENGINE_TASK_TIMEOUT 配置）
- 失败重试：3次，间隔1秒

**不同任务类型的默认值**：
- 图像生成：轮询间隔5秒，超时2分钟
- 视频生成：轮询间隔15秒，超时5分钟
- 音频生成：轮询间隔5秒，超时2分钟

### 任务管理功能

1. **创建任务**：自动记录任务ID和元数据
2. **查询任务**：
   - 查询单个任务状态
   - 列出所有任务（支持按状态、时间筛选）
3. **取消任务**：删除正在运行的任务
4. **下载结果**：
   - 下载到本地（指定路径）
   - 获取URL（直接访问）
5. **任务历史**：
   - 记录所有任务
   - 支持引用之前的任务结果

### 任务状态映射

```
API状态 → Skill显示
- queued → "排队中"
- running → "生成中..."
- succeeded → "已完成 ✓"
- failed → "失败 ✗"
- cancelled → "已取消"
```

### 错误处理

**轮询过程中的错误**：
- 网络错误：重试3次，指数退避（1s, 2s, 4s）
- API限流：等待60秒后重试
- 任务失败：返回错误信息，引导用户重试

**超时处理**：
```
Skill：任务超时（5分钟内未完成）
      任务ID: task-12345
      您可以：
      1. 继续等待（延长超时）
      2. 稍后查询
      3. 取消任务
```

## 问题6：错误处理和用户体验 ✅ 已确认

### 错误信息展示 - 友好提示+解决方案

**原则**：
- 永远不显示原始技术错误
- 每个错误都提供解决方案
- 尽可能提供替代方案
- 引导用户修复问题

**错误信息模板**：

#### API认证错误
```
❌ 认证失败

您的API Key可能无效或已过期。

解决方案：
1. 检查环境变量 ARK_API_KEY 是否设置正确
2. 访问 https://console.volcengine.com/ark/apikey 获取新的API Key
3. 设置环境变量：export ARK_API_KEY=your-key

需要我帮您配置API Key吗？
```

#### 余额不足
```
❌ 余额不足

您的账户余额不足以完成此操作。

解决方案：
1. 访问控制台充值：https://console.volcengine.com/ark/billing
2. 减少生成内容的大小或时长
3. 等待套餐重置（如有免费额度）

充值后可以直接重试。
```

#### 参数错误
```
❌ 参数错误：时长超出范围

您设置的时长：100秒
允许范围：1-60秒

建议：
• 使用默认值：5秒
• 设置合理值：--duration 10

您希望使用哪个时长？
```

#### 内容审核失败
```
❌ 内容生成失败：内容审核不通过

可能的原因：
• 提示词包含敏感内容
• 图片内容不符合规范

替代方案：
1. 修改提示词，去除敏感词汇
2. 更换参考图片
3. 尝试不同的镜头描述

需要我帮您调整吗？
```

#### 网络错误
```
❌ 网络连接失败

无法连接到火山引擎服务器。

自动重试中...（第1/3次）

[如果重试失败]
多次重试后仍然失败。

解决方案：
1. 检查网络连接
2. 检查防火墙设置
3. 稍后再试

是否保存当前任务，稍后继续？
```

### 自动重试策略

#### 网络错误
- 重试次数：3次
- 重试间隔：指数退避（1秒、2秒、4秒）
- 用户提示：显示重试进度

#### API限流（429 Too Many Requests）
- 重试次数：2次
- 重试间隔：60秒
- 用户提示：告知等待时间

#### 任务失败
- 不自动重试
- 显示详细失败原因
- 提供"重试"按钮

#### 服务器错误（5xx）
- 重试次数：2次
- 重试间隔：5秒
- 失败后建议稍后再试
