# 火山引擎通用API操作 Skill

这是一个用于操作火山引擎API的通用Skill，支持图像生成、视频生成和视觉理解。

## 功能列表

### 1. 图像生成 (Seedream 4.0)
- 文本生成图片 (Text-to-Image)
- 图片编辑 (Image Editing)
- 图生图 (Image-to-Image)

### 2. 视频生成 (Seedance 1.5)
- 文本生成视频 (Text-to-Video)
- 图片生成视频 (Image-to-Video)
- 控制镜头运动

### 3. 视觉理解
- 图像内容分析
- 对象检测

### 4. 任务管理
- 查看生成进度
- 下载结果
- 管理历史记录

### 5. 配置设置
- 设置API Key
- 调整默认参数
### 5. 任务管理
- 查看生成进度
- 下载结果
- 管理历史记录

### 6. 配置设置
- 设置API Key
- 调整默认参数

## 参数说明

### 图像生成参数
- prompt (必填): 图片描述
- width (可选): 宽度，默认1024
- height (可选): 高度，默认1024
- negative_prompt (可选): 负向提示词

### 视频生成参数
- prompt (必填): 视频描述
- duration (可选): 时长，1-10秒，默认5
- aspect_ratio (可选): 宽高比，默认"16:9"

## 配置

### 环境变量
- `ARK_API_KEY`: 火山引擎API密钥

### 配置文件
项目配置: `.volcengine/config.yaml`
全局配置: `~/.volcengine/config.yaml`

## 注意事项

1. API Key 是必需的，请先设置环境变量或配置文件
2. 图像尺寸建议使用64的倍数以获得最佳效果
3. 视频时长限制在1-10秒
4. 所有任务都是异步的，可以查看进度

## 错误处理

如果遇到错误，Skill会提供友好的错误信息和解决方案建议：
- 认证错误：检查API Key设置
- 速率限制：等待后重试
- 网络错误：检查网络连接
- 参数错误：检查参数格式
